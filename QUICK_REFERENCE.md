## Chart Types Quick Guide

| Chart Type | What It Shows | How to Read |
|------------|--------------|-------------|
| **Bar Chart** | Comparing categories or time periods | Taller bars = larger values |
| **Line Chart** | Trends over time | Upward slope = increasing trend |
| **Pie Chart** | Parts of a whole | Larger slice = larger share |
| **Area Chart** | Volume/cumulative values | Larger area = greater volume |
| **Radar Chart** | Multi-dimensional comparison | Larger area = better performance |
| **Combined Chart** | Different data types together | Compare bars to lines |

## Key Metrics Explained

### Budget Module
- **Utilization Rate**: % of budget spent (e.g., 95% = 95% spent)
- **Remaining Budget**: Amount left (negative = overrun)
- **Variance**: Difference between planned and actual

### Project Module
- **Completion Rate**: % of projects completed
- **In Progress**: Projects actively being executed
- **On Hold**: Projects temporarily paused

### Program Module
- **Achievement Rate**: % of target achieved
- **Beneficiaries**: People served by programs
- **Satisfaction Score**: Stakeholder satisfaction (0-100)

### Socio-Economic Module
- **Literacy Rate**: % of population that can read/write
- **Enrollment**: % of school-aged children in school
- **Health Score**: Aggregate health indicator (0-100)
- **GDP Growth**: Economic growth rate (%)

## Common Tasks

### Finding Budget Variances
1. Go to **Budget Implementation Analysis**
2. Check "Remaining Budget" metric (red = overrun)
3. Review monthly bar chart for specific months
4. Examine line chart for trend

### Checking Project Status
1. Go to **Project Implementation Analysis**
2. Review pie chart for overall status
3. Check table for detailed status
4. Use sector chart to compare performance

### Assessing Program Impact
1. Go to **Program Tracking**
2. Review beneficiary growth chart
3. Check achievement vs. target chart
4. Review program performance table

### Tracking Development Progress
1. Go to **Social Economic Data**
2. Review population and education trends
3. Check health indicators vs. targets
4. Examine economic growth trends
